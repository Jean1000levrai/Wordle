# Wordle
---

A simple Wordle-style word guessing game built with React. Through this project I will about react, deepen my knowledge of javascript and data implementation in browser

> 🚧 **This project is under active development**. Features and behavior may change.


## 🧩 Features (Planned & Current)

- ✅ User input via on-screen or physical keyboard
- 🚧 Word validation and attempt tracking
- 🚧 Colored feedback (green/yellow/gray) for each guess
- 🚧 history from day to day
- ✅ Word list integration (dictionary check)
- 🚧 Mobile-friendly layout
- 🚧 Dark mode toggle


## 🛠️ Technologies

- **React** (via Vite)
- **JavaScript**
- **CSS**
- Font Awesome (for icons)
